### Authentication

Every API request requires an API key for authentication. Always include your API key in the request header:

<CodeGroup>

```bash cURL
curl --request GET \
  --url https://api.parceltracer.app/v1/external/{endpoint} \
  --header 'X-Api-Key: <api-key>'
```

```python Python
import requests

url = "https://api.parceltracer.app/v1/external/{endpoint}"
headers = {
    "X-Api-Key": "<api-key>"
}

response = requests.get(url, headers=headers)
```

```javascript JavaScript
const response = await fetch('https://api.parceltracer.app/v1/external/{endpoint}', {
  method: 'GET',
  headers: {
    'X-Api-Key': '<api-key>'
  }
});
```

</CodeGroup>
